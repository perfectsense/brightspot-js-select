What's this?

A jQuery plugin so that <select> element doesn't have to look like a <select> element, unless you're on IE8 or older.

And if you're on a mobile device, you're <select> looks the way you want but behaves like a browser's standard <select>.

Notes:

  • Use custom-select.less to customize the look & feel!!
  • The JavaScript is intended to pass events through to the custom UI
